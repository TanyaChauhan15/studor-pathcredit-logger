# PathCredit Logger

Built as part of Studor Builder Screening Project.

A simple React web app to log student activities and view them in a filtered activity feed.

---

## 🚀 How to run locally

1. Install dependencies:
   npm install

2. Start the app:
   npm run dev

3. Open the localhost link shown in terminal

---

## 📌 What I built

I built a minimal PathCredit Logger with:

- A form to log activities (name, category, date)
- An activity feed that updates instantly without page reload
- Category-based filtering
- localStorage persistence across refresh

---

## ⚠️ Edge cases handled

- Prevents submission if any field is empty
- Shows message when no activities are logged
- Shows message when filter returns no results
